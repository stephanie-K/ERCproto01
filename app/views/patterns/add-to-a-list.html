Add to a list

HMRC Design SystemLink for this pattern

This pattern lets users add information with more than one part to a list. This would be useful to add people on a form example.

How it works

Allow users to:

  • view a summary of what they have already added to the list
  • answer questions about what they are adding to the list
  • check their answers before adding to the list
  • delete and change things on the list
